## 📱 HOW TO ACTIVATE/MANAGE

### Via Mobile App
1. Download "Banglalink" app
2. Log in or register
3. Go to "Packages" section
4. Select desired package
5. Tap "Buy" or "Activate"
6. Confirm purchase
7. Receive confirmation SMS

### Via USSD Codes
- **Main Menu:** Dial *123#
- **Buy Package:** *123*[code]#
- **Check Balance:** *111#
- **Data Balance:** *124#
- **Account Info:** *123*0#

### Via Call Center
- **Dial:** 111 (free)
- **Say:** Package name
- **Follow:** IVR instructions
- **Confirm:** Via SMS or app

### Via Website
- Visit: www.banglalink.com.bd
- Log in: Account section
- Browse: Package offerings
- Purchase: Via online payment
- Receive: Confirmation email
